# 本仓库用于学习 NestJS 
如果你想阅读本仓库的内容，可以搭配我的博客使用 **https://mxte.cc/313.html** <br>
推荐的教程 **https://nestjs.bootcss.com/**

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
